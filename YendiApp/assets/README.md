# Assets Directory

Cette directory contient les assets nécessaires pour l'application Yendi.

## Fichiers nécessaires

Pour une application Expo complète, les fichiers suivants sont requis :

- **icon.png** : Icône de l'application (1024x1024 px recommandé)
- **splash.png** : Écran de démarrage
- **adaptive-icon.png** : Icône adaptative pour Android (1024x1024 px)
- **favicon.png** : Favicon pour le web (48x48 px minimum)

## Note

Pour le moment, Expo peut fonctionner sans ces fichiers en mode développement et utilisera des assets par défaut.
